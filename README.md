# ChatBox

Simple single-user Electron GUI for cloud and local LLM's

![screenshot](screenshot.png)

## Features

- Multiplatform
    - macOS
    - Windows
    - Linux
- OpenAI compatible providers support
    - OpenRouter
    - Ollama
    - AI Studio (text-only)
    - many others...
- Multimodality support
    - Image
    - Audio
    - Video
- No Docker required

## Running locally

```bash
npm install
npm run electron:dev
```
